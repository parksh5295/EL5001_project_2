python -m threat_agent.stream_labeler --input events.ndjson --output results/events_weak_labeled.ndjson
준-현실형 스트림 생성
python -m threat_agent.stream_builder --input results/events_weak_labeled.ndjson --output results/stream_events.ndjson

python -m threat_agent.stream_baselines --stream-data results/stream_events.ndjson --split test --episodes 100 --policy both
스트림 RL 학습(DQN)
python -m threat_agent.train_stream_dqn --stream-data results/stream_events.ndjson --episodes 1500
스트림 RL 학습(REINFORCE)
python -m threat_agent.train_stream_reinforce --stream-data results/stream_events.ndjson --episodes 1500
스트림 RL 학습(A2C)
python -m threat_agent.train_stream_a2c --stream-data results/stream_events.ndjson --episodes 1500
스트림 RL 학습(Tabular: MC/SARSA/Q-learning)
python -m threat_agent.train_stream_tabular --stream-data results/stream_events.ndjson --algorithm all --episodes 3000 --output results/stream_tabular_metrics.json
